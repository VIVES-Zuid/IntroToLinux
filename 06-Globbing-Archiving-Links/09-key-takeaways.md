# Key Takeaways


- Globbing patterns make file selection efficient and powerful
- tar combines archiving and compression for backup strategies
- Understanding links helps with file system organization
- Inodes are fundamental to Unix file systems
- dd is powerful but requires careful usage
- Practice with safe test data before using on important files
