# 5. Home Directory Concepts


### The /home Directory Structure
![Linux File System Homes](../images/LinuxFileSystemHomes.png)

Every user gets their own directory under `/home/username`. This is where users can:
- Create files and folders
- Store personal data
- Install user-specific programs

### User's Home Directory
![Linux File System Home](../images/LinuxFileSystemHome.png)

Your home directory (`/home/username`) is:
- Referenced by `$HOME` environment variable
- Accessible via `~` shortcut
- Your default location when logging in
- Only place where you have full write permissions
