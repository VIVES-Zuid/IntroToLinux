# Lesson 12: Exam Preparation Recap

This chapter is a targeted review of the most exam-relevant Linux commands and concepts from lessons 1–10. Each section gives a brief explanation of the concept, the key syntax, and a set of practice exercises.

> ⚠️ The exam is **open book** but **time-limited**. Speed comes from knowing *which* command to reach for and *how* to combine them — not from memorising man pages.

Attempt every exercise before checking the [solutions](../Solutions/lesson-12-recap-solutions.md).
